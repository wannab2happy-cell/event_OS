## 1. 로그인

### Magic Link 로그인

1. **이벤트 랜딩 페이지 접속**
   - URL: `https://[your-domain].vercel.app/[eventId]`
   - 이벤트 초대 메일에서 받은 링크 사용

2. **등록 시작**
   - "등록 정보 입력하기" 버튼 클릭
   - 또는 URL: `https://[your-domain].vercel.app/[eventId]/register`

3. **이메일 입력**
   - 초대받은 이메일 주소 입력
   - "Magic Link 발송" 버튼 클릭

4. **Magic Link 확인**
   - 이메일 수신함에서 Magic Link 확인
   - Magic Link 클릭하여 자동 로그인

### 로그인 후

- 로그인 성공 시 자동으로 등록 페이지로 이동
- 진행 중인 등록 단계부터 계속 진행 가능

## 2. 등록 절차

### 등록 단계 개요

참가자 등록은 다음 4단계로 구성됩니다:

1. **기본 정보** (basic-info)
2. **여권 정보** (passport)
3. **항공 정보** (flight)
4. **호텔 정보** (hotel)

### 1단계: 기본 정보 입력

1. **필수 정보 입력**
   - **이름** (필수): 한글 또는 영문 이름
   - **이메일** (필수): 초대받은 이메일 주소
   - **연락처** (필수): 전화번호 (예: 010-0000-0000)

2. **선택 정보 입력**
   - **회사**: 소속 회사명
   - **직책**: 직책 또는 부서

3. **저장**
   - "저장" 버튼 클릭
   - 저장 성공 시 자동으로 다음 단계로 이동

### 2단계: 여권 정보 입력

1. **여권 정보 입력**
   - **여권 번호** (필수): 여권 번호 입력
   - **여권 만료일** (필수): YYYY-MM-DD 형식
   - **비자 필요 여부**: 비자가 필요한 경우 체크

2. **저장**
   - "저장" 버튼 클릭
   - 저장 성공 시 자동으로 다음 단계로 이동

### 3단계: 항공 정보 입력

1. **개인 정보 입력**
   - **성별**: 남성 또는 여성 선택
   - **생년월일**: YYYY-MM-DD 형식
   - **좌석 선호도**: 창가, 복도, 중간 등

2. **도착 정보 입력**
   - **도착일**: YYYY-MM-DD 형식
   - **도착 시간**: HH:MM 형식
   - **도착 공항**: 공항 코드 또는 공항명
   - **도착 항공편**: 항공편 번호

3. **출발 정보 입력**
   - **출발일**: YYYY-MM-DD 형식
   - **출발 시간**: HH:MM 형식
   - **출발 공항**: 공항 코드 또는 공항명
   - **출발 항공편**: 항공편 번호

4. **저장**
   - "저장" 버튼 클릭
   - 저장 성공 시 자동으로 다음 단계로 이동

### 4단계: 호텔 정보 입력

1. **호텔 정보 입력**
   - **체크인 날짜**: YYYY-MM-DD 형식
   - **체크아웃 날짜**: YYYY-MM-DD 형식
   - **객실 선호도**: 싱글 또는 트윈 선택
   - **룸 쉐어링 정보**: 동반자 정보 (필요시)

2. **저장 및 완료**
   - "저장" 버튼 클릭
   - 저장 성공 시 등록 완료
   - QR PASS 페이지로 이동 가능

### 등록 진행 상황 확인

- 상단에 진행 상황 표시줄이 표시됩니다
- 완료된 단계는 체크 표시
- 현재 단계는 강조 표시
- 미완료 단계는 회색 표시

## 3. QR PASS 확인

### QR PASS 페이지 접속

1. **QR PASS 페이지로 이동**
   - 등록 완료 후 자동으로 이동
   - 또는 URL: `https://[your-domain].vercel.app/[eventId]/qr-pass`

2. **QR 코드 확인**
   - 화면 중앙에 QR 코드 표시
   - 행사 당일 현장 체크인 시 사용

3. **참가자 정보 확인**
   - 이름, 회사, 이메일 정보 표시
   - 등록 상태 표시

### QR PASS 사용 방법

1. **행사 당일**
   - QR PASS 페이지를 스마트폰으로 열기
   - 또는 QR 코드를 스크린샷으로 저장

2. **현장 체크인**
   - 현장에서 QR 코드를 스캔
   - 체크인 완료

### QR PASS 정보

QR PASS에는 다음 정보가 포함됩니다:

- 참가자 이름
- 참가자 이메일
- 이벤트 ID
- 참가자 ID
- 등록 상태

## 4. 확정 정보 확인

### 항공 확정 정보

Admin이 항공 확정 정보를 입력하면:

1. **확정 메일 수신**
   - 이메일로 확정 메일 자동 발송
   - 발권 번호 (E-Ticket No) 포함

2. **QR PASS 페이지에서 확인**
   - QR PASS 페이지에 항공 확정 정보 표시
   - 발권 번호 확인
   - 도착/출발 정보 확인

### 호텔 확정 정보

Admin이 호텔 확정 정보를 입력하면:

1. **확정 메일 수신**
   - 이메일로 확정 메일 자동 발송
   - 확정 번호 (Confirmation No) 포함

2. **QR PASS 페이지에서 확인**
   - QR PASS 페이지에 호텔 확정 정보 표시
   - 확정 번호 확인
   - 체크인/체크아웃 날짜 확인

### 확정 정보 확인 방법

1. **QR PASS 페이지**
   - `/[eventId]/qr-pass` 접속
   - 확정 정보 섹션 확인

2. **확정 메일**
   - 이메일 수신함에서 확정 메일 확인
   - 메일 내용에 확정 정보 포함

## 5. 문제 해결

### 로그인 문제

#### Magic Link를 받지 못했어요

1. **이메일 확인**
   - 스팸 폴더 확인
   - 이메일 주소 오타 확인

2. **재시도**
   - 등록 페이지에서 이메일 재입력
   - "Magic Link 발송" 버튼 다시 클릭

3. **문의**
   - 이벤트 주최측에 문의

#### Magic Link가 작동하지 않아요

1. **링크 확인**
   - 링크가 만료되지 않았는지 확인
   - 링크를 복사하여 새 탭에서 열기

2. **브라우저 확인**
   - 다른 브라우저에서 시도
   - 시크릿 모드에서 시도

3. **문의**
   - 이벤트 주최측에 문의

### 등록 문제

#### 정보를 저장할 수 없어요

1. **필수 항목 확인**
   - 모든 필수 항목이 입력되었는지 확인
   - 이메일 형식이 올바른지 확인

2. **네트워크 확인**
   - 인터넷 연결 상태 확인
   - 페이지 새로고침 후 재시도

3. **문의**
   - 이벤트 주최측에 문의

#### 등록 진행 상황이 저장되지 않아요

1. **브라우저 확인**
   - 브라우저 캐시 삭제
   - 쿠키 삭제 후 재로그인

2. **재시도**
   - 정보를 다시 입력하여 저장
   - 각 단계를 순서대로 진행

3. **문의**
   - 이벤트 주최측에 문의

### QR PASS 문제

#### QR 코드가 표시되지 않아요

1. **로그인 확인**
   - 올바르게 로그인되었는지 확인
   - 재로그인 시도

2. **등록 완료 확인**
   - 모든 등록 단계가 완료되었는지 확인
   - 미완료 단계가 있으면 완료

3. **문의**
   - 이벤트 주최측에 문의

#### QR 코드를 스캔할 수 없어요

1. **화면 밝기 확인**
   - 화면 밝기를 최대로 설정
   - 반사 방지

2. **QR 코드 크기 확인**
   - QR 코드가 충분히 크게 표시되는지 확인
   - 확대하여 표시

3. **스크린샷 사용**
   - QR 코드를 스크린샷으로 저장
   - 스크린샷으로 스캔 시도

### 확정 정보 문제

#### 확정 메일을 받지 못했어요

1. **이메일 확인**
   - 스팸 폴더 확인
   - 이메일 주소 확인

2. **확정 상태 확인**
   - QR PASS 페이지에서 확정 상태 확인
   - Admin이 확정 정보를 입력했는지 확인

3. **문의**
   - 이벤트 주최측에 문의

#### 확정 정보가 표시되지 않아요

1. **확정 상태 확인**
   - QR PASS 페이지에서 확정 상태 확인
   - Admin이 확정 정보를 입력했는지 확인

2. **페이지 새로고침**
   - 페이지 새로고침
   - 재로그인 후 확인

3. **문의**
   - 이벤트 주최측에 문의

## 6. 자주 묻는 질문 (FAQ)

### Q: 등록을 완료했는데 QR PASS가 보이지 않아요.

A: 모든 등록 단계(기본 정보, 여권, 항공, 호텔)가 완료되었는지 확인하세요. 미완료 단계가 있으면 완료한 후 QR PASS 페이지로 이동하세요.

### Q: 정보를 수정하고 싶어요.

A: 등록 페이지(`/[eventId]/register`)에서 각 단계로 이동하여 정보를 수정할 수 있습니다. 또는 확정 메일의 "등록 정보 수정 바로가기" 링크를 사용하세요.

### Q: 확정 정보는 언제 받을 수 있나요?

A: Admin이 항공 또는 호텔 확정 정보를 입력하면 자동으로 확정 메일이 발송됩니다. 확정 정보는 QR PASS 페이지에서도 확인할 수 있습니다.

### Q: 행사 당일 QR 코드를 잊어버렸어요.

A: 이벤트 랜딩 페이지에서 "등록 정보 입력하기"를 클릭하여 로그인한 후 QR PASS 페이지로 이동하세요. 또는 이벤트 주최측에 문의하세요.

### Q: 여러 이벤트에 참가하는데 어떻게 하나요?

A: 각 이벤트마다 별도의 이벤트 ID가 있습니다. 각 이벤트의 랜딩 페이지에서 개별적으로 등록하세요.
